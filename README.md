# Task 3 - Advanced CSS Styling and Responsive Design

## Objective

To create a visually appealing and responsive multi-page portfolio website using HTML, CSS, and JavaScript.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Features

* Multi-page portfolio website
* Responsive navigation bar
* Home, About, Skills, and Contact pages
* Attractive gradient backgrounds
* Hover effects and transitions
* Mobile-friendly responsive design
* External links to GitHub and LinkedIn profiles
* Interactive button using JavaScript

## Pages Included

* Home Page
* About Page
* Skills Page
* Contact Page

## How to Run

1. Download the project files.
2. Open `index.html` in a web browser.
3. Navigate through the pages using the navigation menu.

## Learning Outcome

This project helped me understand responsive web design, CSS styling techniques, navigation between web pages, and basic JavaScript interactions.
